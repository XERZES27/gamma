## Installation

- [Install: Fvm](https://fvm.app/documentation/getting-started/installation)

 
``` fvm use 3.16.0 ```
``` fvm flutter pub get ```

## Local Run

``` fvm flutter pub run ```

## Local Build Bundle

``` fvm flutter build appbundle ```

## Local Build Apk

``` fvm flutter build apk ```

## Local Build Split Apk

``` fvm flutter build apk --split-per-abi```
